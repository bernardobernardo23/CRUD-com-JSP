MAS AFINAL, OQUE É UM CRUD?

CRUD é um acrônimo para Create(Criar),Read (Ler), Update (Atualizar) e Delete (Excluir)
Esses são os quatro pilares das operações básicas em sistemas que manipulam dados.
Quase todos as empresas precisam de um CRUD para sua admnistração interna,
seja para controle de funcionarios , estoque ou alunos, por exemplo

CRUD - JSP.

E O JSP?

A sigla JSP vem de JavaServerPages, a tecnologia JSP permite que você misture código Java e HTML no mesmo documento,
fazendo assim com que você não precise criar dois documentos distintos para fazer a tela de cadastro de produtos, por exemplo.
No documento JSP se faz a ligação com o Banco de Dados por meio de código Java
